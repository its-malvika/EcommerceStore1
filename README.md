# RedStore - E-Commerce Website

A simple, responsive e-commerce website for fitness and lifestyle products built with HTML and CSS.

## Features

- Clean and modern design
- Fully responsive (desktop, tablet, mobile)
- Product showcase with star ratings
- Customer testimonials section
- Newsletter signup form
- Social media links in footer
- Smooth hover animations

## How to Use

1. Download or extract the project files
2. Make sure the `images/` folder is in the same directory as `index.html`
3. Open `index.html` in your web browser
4. Done! The website will load and work perfectly

## Website Sections

- **Navigation Bar** - Links to Home, Products, About, Contact, Account
- **Hero Section** - Main banner with call-to-action
- **Product Categories** - 3 featured categories
- **Featured Products** - 4 showcase products with ratings and prices
- **Latest Products** - 12 newest products
- **Exclusive Offer** - Special product spotlight (Smart Band 4)
- **Customer Reviews** - 3 customer testimonials with ratings
- **Newsletter Signup** - Subscribe for updates
- **Footer** - Company info, links, and social media

## Responsive Design

The website automatically adapts to different screen sizes:

- **Desktop (> 768px)** - Multi-column layout with full navigation
- **Tablet (600px - 768px)** - 2-column product grid
- **Mobile (< 600px)** - Single column layout, touch-friendly
- **Extra Small (< 400px)** - Optimized for very small screens

## Browser Support

Works on:
- Chrome
- Firefox
- Safari
- Edge
- Mobile browsers (iOS Safari, Chrome Mobile)

## Quick Customization

### Change Product Information
Edit product details directly in `index.html`. Find the product cards and update:
- Product image file name
- Product title
- Product description
- Price

### Update Company Information
Edit the footer section in `index.html` to update:
- Company description
- Your name
- Contact information
- Social media links

### Modify Colors
The primary color is `#ff523b`. Search and replace this in `style.css` to change the theme color throughout the website.

## Key Features Explained

**Flexbox Layout** - Uses CSS Flexbox for responsive grid system that adapts to screen size

**Semantic HTML** - Uses proper HTML5 elements like `<header>`, `<section>`, `<footer>` for better structure

**Accessibility** - Includes alt text for images and aria labels for better accessibility

**Smooth Animations** - Hover effects and transitions make the site feel interactive and modern
